I'm a computer science student living in Trondheim, Norway. Here are some of my personal projects.

simple-blog

A simple blog and CMS written in the Meteor framework as an exercise.

Sea Battle

A online multiplayer impelementation of Battleship, written as project in a school subject during spring 2015. It's built with libGDX, Node, Express and PostgreSQL.

MeteorChat

A simple chat application written in the Meteor framework. It was used an example for how easy it is to build something fast with Meteor, in a presentation I held at JavaZone Academy 2014 in Trondheim.

JavaChat

Another chat application. This one is written in Java, and provides a CLI to the user. It has both a server and client program, and communicates through JSON-objects built with a library called simple-json.

TacoCalculator

Another application written in Meteor. This one calculates how much you need of different taco ingredients depending on how many people who're gonna eat.

iBok.no

Norways largest website for buying and selling used books. Contributed to this project for about two years, as a volunteer at "Studentmediene i Trondheim" (a media company by and for students in Trondheim). The backend's built with PHP using the framework CodeIgniter, and the frontend with JavaScript, jQuery and some Angular.
